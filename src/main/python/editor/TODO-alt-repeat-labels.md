# Alt Repeat Key Editor Labels

## Issue
The column headers are cryptic abbreviations that may confuse users:
- LC = Left Ctrl
- LS = Left Shift
- LA = Left Alt
- LG = Left GUI
- RC/RS/RA/RG = Right modifiers
- Def = Default (use this key when no match)
- Bi = Bidirectional
- Ign = Ignore modifiers

## Current State
Tooltips added to explain each abbreviation. Labels use palette(mid) color, 9px font.

## Potential Improvements
- Longer labels if space allows
- Icons instead of text
- Expandable/collapsible detailed view
- User preference for verbose vs compact mode
